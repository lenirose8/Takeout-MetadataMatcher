# Takeout-MetadataMatcher
Hallo, das hier ist ein einfaches tool um die Metadaten aus eurem Google Takeout zu fixen. Es entnimmt die Metadaten aus der json Datei und integriert sie in euren normalen Bilder und Videos.
Falls ihr gar keine Ahnung von Coden und Programmieren habt, ist das genau eure Anlaufstelle.
